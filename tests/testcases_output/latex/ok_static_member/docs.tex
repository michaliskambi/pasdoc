\item[\textbf{Description}]
All methods are static unless they are virtual or dynamic. but \begin{ttfamily}StaticProcedure\end{ttfamily} is declared static. Even if 'static' did not appear after the declaration, \begin{ttfamily}StaticProcedure\end{ttfamily} would still be static. \begin{ttfamily}StaticProcedure\end{ttfamily} is also a Class procedure but that is a separate story.
